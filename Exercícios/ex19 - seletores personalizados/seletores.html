Seletores Personalizados

As vezes precisamos ter dois itens de mesma classe em estilos diferentes. Para isso, precisamos diferenciar os itens por classes.

id no HTML é # no CSS

id: serve para estilizar apenas uma única tag.

class no HTML é . no CSS

class: serve para estilizar várias tags de mesma classe

span seleciona trexo de texto

span: seleciona trexo do texto ajudando a estilizar com seletores. Exemplo: Básico, Intermediário e Avançado.
