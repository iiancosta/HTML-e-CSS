Colocando Áudio

Nesse site, eu vejo as diferentes maneiras de colocar áudio no html.

Maneira Simples

Usasse as tags: audio e controls.

Maneira Detalhada

Tags usadas:

  • audio
    1. preload serve para informar ao navegador, quanto o site irá carregar do áudio.
      • auto
        • Carrega o arquivo por completo. Obs: Se o áudio for grande deixará o site lento.
      • metadata
        • Carrega apenas o tempo, tamanho e informações de direito.
      • none
        • Não carrega informações do áudio até que seja tocado o play.
    2. autoplay carrega o site e toca o áudio automaticamente.
    3. controls faz com que apareça a interface de interação com o áudio.
    4. loop faz a música se repetir.
  • source
    • Chama o arquivo.
    1. src chama o arquivo.
    2. type informa a extensão do áudio.
      • mp3
      • oog
      • wav
        • É uma extensão de alta qualidade e por isso pesada para se colocar em site.
  • download
    • Em caso do navegador não reconhecer as extensões do Source, recomenda-se disponibilizar o áudio para baixar.
